# Projeto de Deploy Contínuo

Este repositório contém a base para um pipeline de integração contínua e deploy contínuo com GitHub Actions.

## Estrutura do repositório

- `.github/workflows/cd.yml`: workflow principal do pipeline.
- `README.md`: documentação do fluxo e dos ambientes.
- `src/`: diretório base para o código da aplicação.
- `tests/`: diretório base para testes.

## O que será explicado

- como o pipeline funciona;
- o papel de cada ambiente;
- onde ocorre a aprovação manual;
- como os secrets seriam utilizados;
- como visualizar a execução do workflow.
